compare_images(image1, image2, thr=1.0)¶

wrapper function to compare two images if the belong to the same person or not

Args :

image1 (string) : path to first image image2 (string) : path to second image thr (float) : threshold to compare eculidian distance

Returns :

True if the same person appears in the two images and els False string : the message to explain which image doesn’t have a face or ok to indicates no matched faces

process_image(image, model, show=False)¶

wrapper function to recognize the name of the person who appears in an image

Parameters

  • image (string) – the system path to the image (should be all in latin letters)

  • model (string) – the recognition model (knn kmeans kdtree)

Returns

name (string - list) the name of the recognized person or list of names of people who appear in the image

if no face was detected it returns ‘no face detected’

  • align_model (string) – alignment model (2d)

  • prediction_model (string) – prediction model (knn kmeans kdtree)

  • track_model (string) – tracking model (sort)

  • rescale_options (string/array) – if set to auto this will let the library decide how to process input image or provide the width and hieght to rescale the image.

Returns

Returns
